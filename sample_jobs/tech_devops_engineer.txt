DevOps Engineer - Cloud Infrastructure
CloudScale Systems
Austin, TX | Remote

About CloudScale Systems:
We're a cloud infrastructure company providing scalable solutions for enterprise clients. Our platform handles millions of requests daily and maintains 99.99% uptime across global deployments.

Position Summary:
We're seeking an experienced DevOps Engineer to join our Infrastructure team. You'll be responsible for designing, implementing, and maintaining our cloud infrastructure while ensuring high availability, security, and performance.

Core Responsibilities:
- Design and manage cloud infrastructure on AWS, Azure, and Google Cloud
- Implement and maintain CI/CD pipelines using Jenkins, GitLab CI, or GitHub Actions
- Automate infrastructure provisioning using Terraform and CloudFormation
- Monitor system performance and implement alerting solutions
- Ensure security best practices across all environments
- Collaborate with development teams on deployment strategies
- Troubleshoot production issues and implement preventive measures
- Maintain documentation for infrastructure and deployment processes

- Bachelor's degree in Computer Science, Engineering, or related field
- 3-5 years of DevOps or Site Reliability Engineering experience

- Scripting skills in Python, Bash, or PowerShell
- Experience with monitoring tools (Prometheus, Grafana, CloudWatch)
- Understanding of networking concepts and security principles
- Experience with version control systems (Git)
