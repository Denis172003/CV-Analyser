## Usage Instructions

1. **Copy Job Description**: Open any `.txt` file and copy the content
2. **Paste in Application**: Use the job description text area in the AI Resume Analyzer
3. **Test Matching**: Upload a resume and see compatibility scoring
4. **Review Optimization**: Check the generated CV optimization advice

## Expected Outcomes

Each job description is designed to test specific aspects of the matching algorithm:

- **Skill Gap Analysis**: Identify missing technical and soft skills
- **Experience Level Matching**: Entry, mid-level, or senior position alignment
- **Industry Keywords**: ATS-friendly terminology extraction
- **Responsibility Alignment**: Past experience matching job duties
- **Company Culture Fit**: Cultural values and work environment indicators

## Testing Best Practices

1. **Complete Job Descriptions**: Use full job postings with all sections
2. **Varied Resume Types**: Test with different experience levels and backgrounds
3. **Language Testing**: Try both English and Romanian resumes where applicable
4. **Edge Cases**: Test with incomplete or poorly formatted job descriptions
5. **Batch Testing**: Use multiple job descriptions to test consistency
